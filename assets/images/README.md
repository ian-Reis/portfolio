# Imagens dos projetos

Coloque aqui os screenshots reais dos projetos (Cozy Kart, RPG Framework, Roblox Game) e troque os placeholders em `index.html` por tags `<img>`, por exemplo:

```html
<img src="assets/images/cozy-kart.png" alt="Cozy Kart — gameplay screenshot" style="width:100%;height:100%;object-fit:cover;">
```

Cada placeholder atual está marcado com a classe `project-shot` dentro de uma `div` de 190px de altura.
